# Recommended VS Code extensions

We recommend installing the following extensions for VS code to make writing your docs easier.

## Linters

- [markdownlint](https://marketplace.visualstudio.com/items?itemName=DavidAnson.vscode-markdownlint)
- [writegoodlinter](https://marketplace.visualstudio.com/items?itemName=travisthetechie.write-good-linter)
- [OpenAPI-lint](https://marketplace.visualstudio.com/items?itemName=mermade.openapi-lint)

## Spell checker

- [Code Spell Checker](https://marketplace.visualstudio.com/items?itemName=streetsidesoftware.code-spell-checker)

## Markdown tools

- [Markdown All-in-One](https://marketplace.visualstudio.com/items?itemName=yzhang.markdown-all-in-one)

## OpenAPI tools

- [OpenAPI-designer](https://marketplace.visualstudio.com/items?itemName=philosowaffle.openapi-designer)
- [YAML language support](https://marketplace.visualstudio.com/items?itemName=redhat.vscode-yaml)
